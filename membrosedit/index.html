Como usar este editor

À esquerda de cada aula está a sugestão original (em dourado, não editável) — use como referência. À direita está o campo de edição onde você personaliza com os nomes egípcios. Quando terminar, clique em Copiar HTML Completo e cole no seu servidor. O YouTube ID vai no campo ytId — ex: URL youtube.com/watch?v=dQw4w9WgXcQ → ID é dQw4w9WgXcQ.
